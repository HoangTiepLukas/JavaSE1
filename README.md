## Java SE I
This project is a simple Java console application that allows the user to create and manipulate a single integer array.

### Functions
- Create array(randomly or manually)
- Print array
- Add number
- Find max
- Find min
- Remove number
- Delete array
- Exit

### Running the Program
Using IntelliJ IDEA
Open the project.

- Open Main.java.

- Click Run.

### Testing
- JUNIT 5 for testing
